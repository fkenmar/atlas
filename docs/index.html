atlas

Same answers, ~65% fewer tokens.

atlas compiles your repo into a compact, ranked, token-budgeted structural map your AI coding agent reads in one shot — every signature, type, and import edge, no function bodies — so it stops burning context just to find its way around.

20/20 accuracy, −65% tokens, 1 turn instead of 3.
In a comprehension benchmark (20 verified questions on a real repo), agents answered with identical accuracy using ~65% fewer tokens (85,670 → 29,781). Honest caveat: on open-ended edit tasks the token numbers are too noisy to claim — we don't headline what we can't measure. Reproduce it yourself →

Try it in 30 seconds

pipx install --pre atlas-map     # or: pip install --pre atlas-map
cd path/to/your/project
atlas .                          # ranked, ~2,000-token map to stdout
atlas . -o atlas-map.md          # save it for your agent

Why it works

  • Structure only. Signatures, types, and imports — never function bodies. A fraction of the source.
  • Ranked & budgeted. PageRank over the import graph, packed to a token budget, so the map stays small enough to live in context every turn.
  • Local & offline. No network calls, no telemetry, deterministic output.
  • Fits any agent. Markdown / JSON / XML to stdout or a file, a structural diff mode, and a read-only MCP server.
