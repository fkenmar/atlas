how the map is drawn

A cartographer for code.

Most of an agent's token bill on an unfamiliar repo is exploration — opening file after file to build a mental model. A naive dump of every file is too big and costs more than it saves. atlas earns its place two ways:

  • structure only — signatures, types, and imports, never function bodies. A fraction of the source.
  • ranked & budgeted — a PageRank over the import graph scores every file by centrality, then packs the most important into a fixed token budget.

what comes out
Ranked, not dumped. #1 is the most central file by PageRank — the agent reads the load-bearing code first.
Reverse edges, too. used by tells the agent who depends on a file — the thing grep can't show at a glance.
Signatures, never bodies. Everything needed to navigate, nothing it doesn't. ~100× smaller than the source.
Deterministic. Same repo + flags → byte-identical output. Diffs are meaningful; a committed map stays cache-stable.
get it running in a minute

Install

$pipx install --pre atlas-map

For the Python crowd — no Rust required. The wheel just drops the native atlas command onto your PATH (the way ruff and uv ship). The PyPI name is atlas-map; the command stays atlas. --pre is required while atlas is in alpha.

$curl -LsSf https://github.com/fkenmar/atlas/releases/.../atlas-installer.sh | sh

macOS & Linux, no Rust required. Windows binaries (x64 + arm64) are attached to every release by cargo-dist.

$git clone …/atlas && cd atlas && cargo install --path .

Any platform with Rust. Builds atlas into ~/.cargo/bin.

$ atlas --version  ·  cd your/project  ·  atlas .   → a ranked map of your repo. that's the whole tool.

wire it into your agent

Built to be composed.

Pipe it inline

{ echo "Repo map:"; atlas .; } | your-agent

stdout in, agent context out. Works with any CLI agent.

Save & @-mention

atlas . -o atlas-map.md

Drop it into Claude Code, Cursor, Copilot — then reference the file.

Serve over MCP

atlas serve --mcp

Agents pull a fresh map as a tool call — get_map, get_symbol.

Keep it fresh

atlas . --check atlas-map.md

Gate a committed map in CI — exit 1 if it's stale. Diffs stay clean.
